# Fair Road

A simple web page deployed on GitHub Pages.

## Deployment

This project is configured for GitHub Pages. To deploy:

1. Push this repository to GitHub
2. Go to your repository settings on GitHub
3. Navigate to "Pages" in the left sidebar
4. Select the source branch (usually `main` or `master`)
5. Your site will be available at `https://[username].github.io/Fair-Road/`
